Visual Studio and GIT download step by step
1. Download Visual Studio by going to https://code.visualstudio.com/download
2. Download Git by going to https://git-scm.com/install/
3. Open VS Code and click on the accounts icon at the bottom, and click on Sign in to sync settings to choose your GitHub account to link to your profile.

creating and cloning a repository
1. Login into github using Gmail or your school email.
2. Click the + icon in the top right and select new repository.
3. Name the repository A02.
4. Check the add a README file and click create repository.
5. In Visual Studio, click the search bar in the top middle and type Git: Clone, and paste your repository URL
6. Select a local folder to save the project.

glossary
**Branch** - A separate workspace where you can make changes and try new ideas without affecting the main project.
**Clone** - A full copy of a repository, including all logging and versions of files.
**Commit** - A save point in your project where it records a snapshot of your files at a certain time, with a message describing what changed.
**Fetch** - Gets all the change history of a tracked branch/repository.
**GIT** - A popular version control system that is used for tracking code changes, tracking who made changes, and coding collaboration.
**Github** - A code hosting platform for collaboration and version control.
**Merge** - Combine changes from different branches.
**Merge Conflict** - happens when two branches change the same part of a file.
**Push** - Send your changes to  a remote repository.
**Pull** - get the latest changes from a remote repository.
**Remote** - Are references to remote repositories.
**Repository** - A folder where Git tracks your project and its history.
